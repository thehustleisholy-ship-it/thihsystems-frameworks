# Framework 01: Women's Health Longitudinal Dashboard

**Patient-owned. Longitudinal. Clinical-conversation-ready.**

## Thesis

The data model is medicine. The export format is the appointment.

This framework explores how AI, structured data, and patient-owned records can help people prepare better clinical conversations without replacing clinicians or pretending to diagnose.

## Problem

People often track symptoms, pain, cycles, fatigue, mood, labs, medications, and life events at home. But by the time they reach an appointment, that lived data is often compressed into a few rushed minutes.

The data exists.

The system often cannot see it.

## What this framework includes

- Cycle records
- Daily symptom check-ins
- Life event tracking
- Clinical record structure
- Medication and side-effect tracking
- Rule-based pattern detection
- Visit-summary export format
- MVP test plan
- Open questions around consent, privacy, safety, and dignity

## What this is not

This is not medical advice.  
This is not a diagnosis engine.  
This is not a replacement for licensed clinicians.  
This is not a finished product.  
This is not a copy-and-paste mandate.

## Folder contents
