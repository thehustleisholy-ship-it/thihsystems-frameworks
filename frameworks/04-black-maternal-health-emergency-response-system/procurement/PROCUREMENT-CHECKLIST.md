# Procurement Checklist

- [ ] Intended use and prohibited use approved.
- [ ] Obstetric, emergency-medicine, nursing, patient-advocate, accessibility, and human-factors review.
- [ ] HIPAA role/data-flow analysis and BAAs where required.
- [ ] FTC HBNR, state health-data/privacy, breach, recording-consent, and consumer-protection review.
- [ ] FDA function-by-function assessment.
- [ ] Data minimization, encryption, access control, audit logging, deletion, export, correction, and incident response.
- [ ] No advertising or unapproved tracking on sensitive surfaces.
- [ ] Clinical escalation ownership, hours, response expectations, handoff, downtime, and failure modes.
- [ ] Evidence provenance and chain-of-custody design reviewed without promising admissibility.
- [ ] Independent security assessment and accessibility testing.
- [ ] Exit criteria and harm reporting defined before any pilot.
